# getEmittableProperties(properties) → {Object}

Get all of the properties of this entity that can be emitted to clients. This method should be overwritten on each subclass (and any further subclasses), and then called on the superclass of every subclass, calling it's way back up to Entity. So if Player.getEmittableProperties is called, it adds the relevant properties from Player, then adds from Character, and so on until Entity, then returns the result back down the stack.

Parameters:
Name Type Description
properties Object

The properties of this entity that have been added so far. If this is the start of the chain, pass in an empty object.
