# checkStock(index, itemTypeNumber, price) → {Boolean}

Check that the item that a player wants to buy is valid. If someone else buys the same item before they do, the ones next along will be moved along, but their client might not have been told that, so they could end up buying the wrong thing. So check the index of the thing to buy, the type, and the price, to make sure they are the same as what the player sees.

Parameters:
Name Type Description
index Number

The index of the item in the clients stock list for this shop.
itemTypeNumber String

The item on display.
price Number

How much it is on display for.
